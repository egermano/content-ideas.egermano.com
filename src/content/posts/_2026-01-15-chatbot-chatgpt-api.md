## Experiment plans
### Experiment 1 — Build a "Study Buddy" Chatbot
- Objective: Use the ChatGPT API to build a chatbot that can help students with their homework.
- Stack: Python or Node.js, the OpenAI API.
- Steps:
  1. Set up a new project and install the OpenAI library.
  2. Create a simple command-line interface for the chatbot.
  3. Use the ChatGPT API to generate responses to user queries.
  4. Add a system prompt to give the chatbot a personality and a specific set of instructions.
- Expected: The chatbot can answer a variety of questions on a specific topic (e.g., history, science).
- Time est: 2–4 hours
- Difficulty: very low
- Safety/privacy/licensing: Be mindful of the data you're sending to the OpenAI API and ensure it's handled securely.

## Validation checklist
- Technical: Test the chatbot with a variety of questions to ensure it's providing accurate and helpful answers.
- SEO quick tests: A/B test different chatbot personalities to see which one is more engaging.
- Success metrics: A high level of user engagement, positive feedback from users.
