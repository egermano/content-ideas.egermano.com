### Experiment 1 — Code Generation Benchmark
- Objective: Measure the accuracy and relevance of code suggestions from each tool for a variety of common web development tasks.
- Stack: VS Code, with extensions for GitHub Copilot, Tabnine, and Amazon CodeWhisperer.
- Steps:
  1. Create a standardized set of prompts for each tool (e.g., "create a React component for a login form," "write a function to fetch data from an API").
  2. Record the code generated by each tool for each prompt.
  3. Evaluate the generated code for correctness, completeness, and adherence to best practices.
