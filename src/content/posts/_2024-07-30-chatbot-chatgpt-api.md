## Experiment plans

### Experiment 1 — Basic Chatbot Implementation

- Objective: Build a functional chatbot that can answer simple questions about a given topic.
- Stack: Next.js, Vercel AI SDK, OpenAI API.
- Steps:
  1. Set up a new Next.js project.
  2. Install the Vercel AI SDK and the OpenAI client library.
  3. Create a simple UI for the chatbot.
  4. Implement the backend to handle requests to the OpenAI API.
- Expected: A working chatbot that can be deployed to Vercel.
- Time est: 3-5 hours
- Difficulty: low
- Safety/privacy/licensing: Be sure to keep the OpenAI API key secret and to not log any user conversations.
