## Experiment plans

### Experiment 1 — Build a chatbot for a documentation website

- Objective: To build a chatbot that can answer questions about a specific documentation website.
- Stack: Node.js, a vector database like Pinecone, LangChain, and a documentation website to scrape.
- Steps:
  1. Scrape the documentation website to create a knowledge base.
  2. Use a vector database to index the knowledge base.
  3. Use LangChain to build a RAG-powered chatbot that can answer questions about the knowledge base.
- Expected: The chatbot can accurately answer questions about the documentation website.
- Time est: 4–6 hours
- Difficulty: medium
- Safety/privacy/licensing: Make sure you have the right to scrape the documentation website.
